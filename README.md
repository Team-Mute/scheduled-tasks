# 프로젝트 소개

<img width="1920" height="1080" alt="프로젝트 대표 이미지" src="https://github.com/user-attachments/assets/bd6f809d-c0c4-4623-a4f2-62cc8eeaca2d" />

- 신한금융희망재단의 **공간 예약 서비스 배치 스케줄러 코드**입니다. 
- 예약 상태를 자동으로 갱신하기 위한 정기 실행 스크립트를 포함합니다.  
- 윈도우 또는 리눅스 환경에서 실행할 수 있으며,  
  예약의 `이용 완료`, `반려 처리`를 자동화하여 관리자의 수동 업무를 최소화합니다.

---

# 스케줄러 목적
| 구분 | 조건 | 결과 |
| ---- | ---- | ---- |
| **이용 완료 자동화** | 상태가 `최종 승인 완료`이며, **예약 종료 시간이 현재 시간보다 지난 경우** | 예약 상태를 `이용완료`로 변경 |
| **반려 자동화** | 상태가 `1차 승인 대기` 또는 `2차 승인 대기`이며, **예약 날짜가 오늘 또는 지난 경우** | 예약 상태를 `반려`로 변경 |

- 🕒 해당 스케줄러는 리눅스의 `cron` 또는 윈도우의 `Task Scheduler`를 통해 주기적으로 실행되며,  
SQL 스크립트를 호출하여 예약 상태를 자동 업데이트합니다.
- 실행 방법은 별도로 제공된 개발 문서를 참고해 주세요.

---

# 사용 기술

| 구분 | 기술 |
| ---- | ---- |
| **Language** | Shell Script, Batch Script |
| **Database** | PostgreSQL |
| **Scheduler** | Linux `cron`, Windows Task Scheduler |
| **Script Management** | `.sh`, `.bat` 파일 기반 실행 구조 |
| **SQL Execution** | psql 명령어를 통한 쿼리 실행 |

---

# 폴더 구조

```plaintext
Team-Mute-backend-scheduler
├── cron                # 리눅스 환경용 크론 설정 파일 (주기적 실행 스케줄 등록)
├── linux_sh            # 리눅스 환경용 실행 스크립트 (.sh 파일)
├── window_bat          # 윈도우 환경용 실행 스크립트 (.bat 파일)
├── sql                 # 배치 스케줄러 실행 시 호출되는 SQL 스크립트
└── .gitignore

```

### 폴더 컨벤션
- **cron/**
  - 리눅스 서버에서 예약 실행을 등록하는 크론탭 설정 파일을 저장합니다.  

- **linux_sh/**
  - 리눅스 환경에서 SQL을 자동 실행시키는 Shell Script 파일을 포함합니다.

- **window_bat/**
  - 윈도우 환경에서 스케줄러를 실행하기 위한 Batch Script 파일을 포함합니다.

- **sql/**
  - 예약 상태 변경을 수행하는 SQL 쿼리 스크립트를 저장합니다.
